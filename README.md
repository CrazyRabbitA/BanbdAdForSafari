# BanBaiduAdForSafari
Safari浏览器禁止百度广告扩展程序

##安装

**注意：**
- Safari浏览器插件必须依赖于提供插件的软件，安装软件后，软件中的插件才会出现在Safari浏览器中。因此，删除该软件，该插件会一并被删除。
- 插件可独立于应用直接在Safari浏览器上运行，使用时无需打开应用，请放心使用。
- 该扩展程序只是隐藏了百度广告，不联网，不会获取网页搜索信息，账号，密码等私密信息。
- 软件开源
